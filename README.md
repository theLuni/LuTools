# LuTools v4.0 – Удаленное управление ПК

Инструмент для удаленного управления компьютером через Telegram-бота.

## Возможности

### Система
- Управление процессами (просмотр, завершение)
- Мониторинг активности пользователя
- Статистика системы (CPU, память, диск)

### Интерфейс
- Создание скриншотов
- Отправка уведомлений на рабочий стол
- Работа с файлами

## Быстрый старт

### 1. Установка Python
**Обязательно Python 3.10.5!**
Скачать: https://www.python.org/ftp/python/3.10.5/python-3.10.5-amd64.exe

При установке отметьте "Add Python to PATH".

### 2. Настройка бота
1. Создайте бота через @BotFather
2. Получите токен
3. В файле LuTools.py замените:

BOT_TOKEN = '8317044568:AAGv40EWvS5Bli-kmg6Vb-7iyq8E8Lntufs'  // ваш токен
ADMIN_ID = 5929120983  // ваш ID Telegram

### 3. Установка
install.bat
Скрипт проверит Python 3.10.5 и установит библиотеки

### 4. Запуск
start.bat
Бот запустится в фоновом режиме.

## Команды

### Через меню
- Скриншот – снимок экрана
- Менеджер процессов – управление программами
- Мониторинг – статистика активности
- Статистика – информация о системе
- Файловый менеджер – загрузка файлов
- Остановить – выключение бота

## Особенности

### Для процессов
- Автоматические названия программ (Chrome → Google Chrome)
- Игнорирование системных процессов
- Сортировка по использованию памяти

### Для скриншотов
- Несколько методов (PowerShell, pyautogui, mss)
- Автоматическое определение лучшего способа
- Оптимизация размера файла

### Мониторинг
- Отслеживание активных приложений
- Время бездействия пользователя
- История активности

## Техническое

### Библиотеки
- aiogram 2.25.1 – Telegram API
- psutil – работа с системой
- mss/pyautogui – скриншоты

### Управление
start.bat – Запуск
stop.bat – Остановка
install.bat – Переустановка

## Важно

1. Только Python 3.10.5! – другие версии не поддерживаются
2. Права администратора – рекомендуются
3. Храните токен в секрете
4. Проверяйте логи в LuTools.log при проблемах

## Поддержка

Проблемы:
1. Убедитесь в версии Python 3.10.5
2. Проверьте токен и ID
3. Запустите с правами администратора
4. Проверьте логи
